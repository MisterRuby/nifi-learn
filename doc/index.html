🚀 Apache NiFi 학습 가이드

데이터 플로우 자동화를 위한 완벽한 가이드

📚 학습 가이드 소개

이 학습 가이드는 Apache NiFi의 핵심 개념부터 고급 기능까지 체계적으로 다룹니다. 실무에서 바로 활용할 수 있는 예제와 베스트 프랙티스를 제공하여, 효율적인 데이터 플로우 시스템을 구축할 수 있도록 도와드립니다.

각 문서는 시각적 다이어그램과 코드 예제를 포함하여 이해를 돕고, 단계별로 학습할 수 있도록 구성되었습니다.

1

Apache NiFi 개요

NiFi의 기본 개념과 핵심 특징을 이해하고, 데이터 플로우 자동화의 기초를 학습합니다.

  • NiFi 소개 및 핵심 개념
  • 주요 특징과 사용 사례
  • 다른 도구와의 비교
  • 빠른 시작 가이드
 2

아키텍처와 구성요소

NiFi의 내부 아키텍처와 핵심 구성요소들의 역할과 동작 방식을 상세히 살펴봅니다.

  • 시스템 아키텍처
  • FlowFile과 Processor
  • 저장소 시스템
  • 스레드 관리 및 스케줄링
 3

프로세서와 플로우 설계

다양한 프로세서의 사용법과 효과적인 데이터 플로우 설계 패턴을 학습합니다.

  • 주요 프로세서 카테고리
  • 플로우 설계 패턴
  • Expression Language
  • 성능 최적화 팁
 4

클러스터링과 확장성

대규모 데이터 처리를 위한 클러스터 구성과 확장 전략을 다룹니다.

  • 클러스터 아키텍처
  • 로드 밸런싱 전략
  • Site-to-Site 프로토콜
  • 고가용성 구성
 5

보안과 베스트 프랙티스

엔터프라이즈급 보안 설정과 운영 환경에서의 베스트 프랙티스를 제공합니다.

  • 인증 및 인가
  • 데이터 암호화
  • 감사 로깅
  • 운영 관리 가이드
 6

Docker로 NiFi 실습환경 구축

Docker를 활용하여 빠르고 효율적인 Apache NiFi 학습 환경을 구성하는 방법을 다룹니다.

  • Docker 기반 NiFi 설치
  • Docker Compose 활용
  • 클러스터 환경 구축
  • 볼륨 마운트와 데이터 영속성
 7

NiFi Registry - 버전관리와 배포

Apache NiFi Registry를 활용한 플로우 버전 관리, 협업, 그리고 배포 자동화를 학습합니다.

  • Registry 개념과 아키텍처
  • 플로우 버전 관리
  • 환경별 배포 전략
  • CI/CD 파이프라인 통합
